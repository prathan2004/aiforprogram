#ออกแบบฐานข้อมูล
ออกแบบฐานข้อมูลระบบข่าวสาร โดยข่าวสารจะมีหมวดหมู่ข่าวสาร ผู้ที่เขียนข่าว เขียน model และ migration file ก็พอ
